# Customer Shopping Behaviour Analysis (SQL & Power BI)

## Overview
- This project analyzes customer shopping behaviour using transactional data to understand spending patterns, subscription impact, product performance, and           customer demographics.
  SQL is used for business analysis and Power BI for interactive visualization.

## Objectives
- Analyze revenue distribution across customers
- Compare subscriber vs non-subscriber behaviour
- Segment customers based on purchase history
- Build an interactive Power BI dashboard

## Tools Used
- SQL (MySQL/PostgreSQL)
- Power BI
- Power Query

## Key Insights
- Subscribed customers show higher value per customer
- Clothing and accessories generate the highest revenue
- Young adult and middle-aged customers contribute most to revenue

## Dashboard
- An interactive Power BI dashboard was created to visualize KPIs, revenue trends, customer segmentation, and category performance.
